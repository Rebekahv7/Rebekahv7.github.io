Hardware Chart

The Hardware chart was the last assignment in the Hardware Module. The hardware pieces I chose were the Motherboard, Hard disk drive, DVD Drive, Power Supply and RAM (Random Access Memory). It was a chart that had five or six hardware components and information on them as well. There are five different catagories which include the name of the item, a blender model, purpose, price, and how delicate each item is. Next year I would like to learn more about each different piece of hardware.
